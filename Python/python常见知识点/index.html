    1 if __name__ == '__main__':的作用

    一个python文件通常有两种使用方法。第一种是作为脚本直接执行，第二种是被import到其他模块中。因此if __name__ == '__main__':的作用就是控制这两种情况执行代码的过程，只有在第一种情况下if __name__ == '__main__':下的代码才会被执行（即文件作为脚本直接执行），而被import到其他模块不会被执行。

    7 Python 如何在Python中打印一个变量的类型

    要在Python中获得一个变量的类型，可以使用内置的type()函数。 基本语法如下: type(object)

    示例:

    # <class 'str'> <--> 字符串
name = "freeCodeCamp"

# <class 'float'> <--> 浮点类型
score = 99.99

# <class 'list'> <--> 列表
lessons =  ["RWD", "JavaScript", "Databases", "Python"]

# <class 'dict'> <--> 字典
person = {
    "firstName": "John",
    "lastName": "Doe",
    "age": 28
}

# <class 'tuple'> <--> 元组
langs = ("Python", "JavaScript", "Golang")

# <class 'set'> <--> 集合
basics = {"HTML", "CSS", "JavaScript"}

print("The variable, name is of type:", type(name))
print("The variable, score is of type:", type(score))
print("The variable, lessons is of type:", type(lessons))
print("The variable, person is of type:", type(person))
print("The variable, langs is of type:", type(langs))
print("The variable, basics is of type:", type(basics))

    输出结果如下所示： The variable, name is of type: <class 'str'> The variable, score is of type: <class 'float'> The variable, lessons is of type: <class 'list'> The variable, person is of type: <class 'dict'> The variable, langs is of type: <class 'tuple'> The variable, basics is of type: <class 'set'>

    8 Python ord()函数

    • 描述 ord()函数是chr()函数(对于8位的ASCII字符串)或unichr()函数(对于Unicode对象)的配对函数，它以一个字符(长度为1的字符串)作为参数，返回对应的ASCII数值，或者Unicode数值，如果所给的Unicode字符超出了Python定义范围，则会引发一个TypeError异常。
    • 语法 ord(c)
    • 参数 c -- 字符
    • 返回值 返回的是对应的十进制整数。

    实例如下所示：

    ord('a')
ord('b')
ord('c')

    上述代码输出为: 97 98 99

    9 Python encode()函数和decode()函数

    encode()方法为字符串(str)提供的方法啊，用于将str类型转换成bytes类型，这个过程也称为“编码”。

    class str(Sequence[str]):
    # -- snip --
    def encode(self, encoding: str = ..., errors: str = ...) -> bytes: ...
    • str 表示要进行转换的字符串。
    • encoding 指定进行编码时采用的字符编码，该选项默认采用utf-8编码。当方法中只使用这一个参数时，可以省略前边的encoding=，直接写编码格式，例如str.encode("utf-8")
    • errors 指定错误处理方式，其可选择值可以是：

      strict：遇到非法字符就抛出异常。 ignore：忽略非法字符。 replace：用“？”替换非法字符。 xmlcharrefreplace：使用 xml 的字符引用。 该参数的默认值为 strict。

    和encode()方法正好相反，decode()方法用于将bytes类型的二进制数据转换为str类型，这个过程也称为“解码”。

    class bytes(ByteString):
    # -- snip --
    def decode(self, encoding: str = ..., errors: str = ...) -> str: ...

    注意： 对 bytes 类型数据解码，要选择和当初编码时一样的格式。

    10 Python str、bytes与bytearray详解

    参考链接: https://blog.csdn.net/u013008795/article/details/88945733

    str: 字符串 bytes: 不可变字节序列 bytearray: 可变的字节数组

    编码与解码
    • 编码: 就是用几个字节去理解内存中的数字 字符串按照不同的字符集编码encode返回字节序列bytes
    str.encode(encoding = 'utf-8', errors = 'strict') -> bytes

    注意: encode默认转换时的默认编码是utf-8

    • 解码: 就是将内存中的字节数按照什么字符集解码成对应的字符
    bytes.decode(encoding='utf-8', errors = 'strict') -> str
bytearray.decode(encoding='utf-8', errors = 'strict') -> str

    注意: decode默认转换时的默认编码是utf-8

    bytes相关方法
    1. 切片语法 1.1 只取出一个元素 bytes[index] 只取出一个元素会变成int类型，如果仍需要使用bytes类型的数据，可以使用bytes()类型转换一下。 1.2 取出某个区间的元素 bytes[index1, index2] [index1, index2) 是个左开右闭区间 1.3 取出开头至某个位置的元素 bytes[:index] [0, index) 是个左开右闭区间 1.4 取出某个位置至末尾的元素 bytes[index:]
    将bytes转换为int

    函数格式:

    int.from_bytes(bytes, byteorder, *, signed=False)

    简单demo:

    s1 = b'\xf1\xff'
print(int.from_bytes(s1, byteorder='big', signed=False))
print(int.from_bytes(s1, byteorder='little', signed=True))

    上述代码的输出为: 61951 -15 参数解释 - bytes 要转换的字节序列 - byteorder 取值为 'big'和'little' 以上例为例，其中big代表正常顺序，即f1ff。little反之，代表反序fff1。 - signed 取值为 True 和 Flase 表示是否要区分二进制的正负数含义。即是否要对原二进制数进行原码反码和补码操作。

    将int转换为bytes

    参考链接: https://vimsky.com/examples/usage/python-int.to_bytes-py.html

    int.to_bytes(length, byteorder, *, signed=False)

    简单demo:

    var = 1024
print(var.to_bytes(2, byteorder='big', signed=False))

var = -1024
print(var.to_bytes(2, byteorder='big', signed=True))

    上述代码输出结果为: b'\x04\x00' b'\xfc\x00'

    参数解释: - length 转换后的字节序列所占字节数 - byteorder 用于表示整数的字节顺序。如果 byteorder 是 "big" ，则最高有效字节位于字节数组的开头。如果 byteorder 是 "little" ，则最高有效字节位于字节数组的末尾。要请求主机系统的本机字节顺序，请使用sys.byteorder 作为字节顺序值。 - signed 确定是否使用二进制补码来表示整数。如果 signed 是 False 并且给定一个负整数，则会引发 OverflowError。 signed 的默认值为 False 。

    11 Python 文件读写

    参考链接: https://zhuanlan.zhihu.com/p/56909212

    Python快速生成指定大小的文件

    参考链接: https://blog.csdn.net/loveangel1/article/details/117323875

    当使用写入空数据的方式生成大文件时，时间消耗是很长的，可以通过seek和write配合的方式快速生成大文件。

    def create_file(file_path, size):
    """
    # 快速生成大文件
    :param file_path: 文件路径
    :param size: 文件大小，本函数以GB为单位，也可以根据需求设置为KB或MB等
    :return:
    """
    # 首先以路径path新建一个文件，并设置模式为写
    lfile = open(file_path, 'w')
    # 根据文件大小，偏移文件写入位置；位置要减掉一个字节，因为后面要写入一个字节的数据
    lfile.seek(1024 * 1024 * 1024 * size - 1)
    # 然后在当前位置写入任何内容，必须要写入，不然文件不会那么大哦
    lfile.write('\x00')     # lfile.write('')不会写入任何内容
    lfile.close()

    12 Python文件操作

    12.1 Python seek()方法

    • 概述 seek()方法用于移动文件读取指针到指定位置。

    • 语法 seek()方法语法如下: fileObject.seek(offset, whence)

    • 参数

      • offset 开始的偏移量，也就是代表需要移动偏移的字节数
      • whence 可选，默认值为0。给offset参数一个定义，表示要从那个位置开始偏移；0代表从文件开头开始算起，1代表从当前位置开始算起，2代表从文件末尾算起。
    • 返回值 如果操作成功，则返回新的文件位置，如果操作失败，则函数返回-1。
    12.2 Python tell()方法

    • 概述 tell()方法返回文件的当前位置，即文件指针当前位置。

    • 语法 tell()方法语法如下: fileObject.tell()

    • 参数 无

    • 返回值 返回文件的当前位置。

    12.3 Python read()方法

    • 概述 read()方法用于从文件读取指定的字节数，如果未给定或为负数则读取所有。

    • 语法 read()方法语法如下: fileObject.read([size])

    • 参数

      • size 从文件中读取的字节数，可选。默认为-1，表示读取整个文件。

    • 返回值 返回从字符串中读取的字节。
