    SVN状态图标的含义

    1. 常规：绿色圈带白色对勾，主要是表示自己电脑上的文件或者文件夹与SVN服务器上对应的文件一致，在自己电脑没有进行任何修改。

    1. 修改：红色圈带白色感叹号，当你开始编辑一个文件，这个文件的状态就会变成修改状态，图标也会变成红色圆圈带白色感叹号。通过这种状态的变化，你就可以知道自己修改过哪些文件，需要提交哪些文件。

    2. 冲突：黄色三角形带黑色感叹号，是指出现了冲突，一般在同时编辑一个文件时容易出现这种冲突的现象。

    3. 只读：灰色圈带白色对勾，这是设置只读属性后才会出现的标识，直到你获得文件锁才能取消只读。只读文件用这个重载图标来表示你必须在编辑之前先得到一个锁。

    4. 已删除：红色叉，表示相应文件或文件夹被计划删除，或者表示文件缺失。

    5. 增加：蓝色加号，表示文件或文件夹已经被计划加入到版本控制之下。通常是新建的文件或文件夹会有这个标识。

    6. 忽略：灰色圈带白色横线，表示已经不受版本控制了。

    7. 无版本控制：蓝色圈带白色问号，表示暂时还没有版本的控制，但没有被忽略。

    8. 锁定：一把黄色的锁，表示文件被锁定，必须先释放锁才能让别人的修改提交。

    svn常用命令

    • 将文件checkout到本地目录 svn checkout path (path是服务器上的目录)

    • 往版本库中添加新文件 svn add file

    • 查看文件或目录状态 svn status path svn status -v path(显示文件和子目录状态)

    • 比较差异 svn diff path(将修改的文件与基础版本比较) svn diff -r m:n path(对版本m和版本n比较差异)

    svn 版本回退

    当我们想放弃对文件的修改，可以使用svn revert命令。 svn revert操作将撤销任何文件或目录里的局部更改。 我们对文件readme进行修改，查看文件状态。
