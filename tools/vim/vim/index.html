    1 Vim 文本操作

    1.1 在正常模式复制，剪切以及粘贴

    当你启动Vim编辑时，默认就进入了正常模式，你可以运行Vim命令，并且浏览整个文件。 从其他任何模式返回正常模式，你只需要按Esc键。 Vim对于复制、剪切、粘贴有它自己的一套术语。复制被叫做yank(y)，剪切被叫做delete(d)，以及粘贴被叫做put(p)。

    1.1.1 复制 (Yanking)

    想要复制文本，将光标放到你想要的地方，然后参考下面的命令按键y。下面是一些有用的命令： - yy 复制当前行，包括换行符 - nyy 复制从光标所在行开始的n行文本 - y^ 复制从光标位置到行首的文本 - y$ 复制从光标位置到行尾的文本 - yw 复制到下一个词的开头 - yiw 复制当前词 - y% 复制匹配符号范围内容。默认支持的符号对是()、{}、[] 这个在复制括号内的内容时，非常有用。光标要放在某一个匹配符号上面。

    1.1.2 全选复制
    • 进入Vim编辑器，按v进入 Visual 模式
    • 键入 gg 到达文本第一行
    • 键入 G 到达文本最后一行并选中所有文本
    • 键入 y 即可复制到寄存器中
    • 键入 p 即可粘贴寄存器中的内容
    1.1.2 剪切 (Deleting)

    在正常模式下，d按键是用来剪切文本的。把光标移动到想要的位置，参考下面的命令按d键： - dd 剪切当前行，包括换行符 - ndd 剪切从光标所在行开始的n行文本 - d^ 剪切从光标位置到行首的文本 - d$ 剪切从光标位置到行尾的文本 - dw 剪切到下一个词的开头 - diw 剪切当前词 - d% 剪切匹配符号范围内容。默认支持的符号对是()、{}、[] 这个在复制括号内的内容时，非常有用。光标要放在某一个匹配符号上面。

    1.1.3 粘贴 (Putting)

    想要粘贴被剪切的内容，先将光标移动到想要的位置，然后按p(小写)可以将内容粘贴到当前光标后面，或者按P(大写)按键可以粘贴到当前光标前面。

    1.1.4 替换 (Replacing)

    在正常模式下，r按键是用来剪切文本。 - 替换光标所在处的字符 r - 替换光标所到之处的字符，直到按下「ESC」键为止。 R

    1.1.5 删除

    通过下面操作删除的字符或行，相当于被剪切掉，可被[p]（粘贴）回来。 - x(小写) 每按一次，删除光标所在位置的一个字符 - nx 例如，6x表示删除光标所在位置后面的6个字符（包括自己在内）。 - X(大写) 大写的X，每按一次，删除光标所在位置前面的一个字符 - nX 例如，20X表示删除光标所在位置的前20个字符（不包括自己在内）。 - dd 删除当前行，包括换行符。 - ndd 删除从光标所在行开始的n行文本。

    1.1.6 撤销
    • u 撤销操作，按多次u可以执行多次撤销。
    • Ctrl + r 撤销的恢复。

    1.2 在可视模式复制，剪切以及粘贴

    在 Vim 中，可视模式用于选择文本，这与在GUI文本编辑器中通过鼠标单击并拖动以选择文本的操作方式等效。

    1.2.1 可视模式下的额外模式

    Vim中的可视模式允许选择文本，但可视模式下不同的子集将很有用，具体取决于要如何选择文本。 Vim中有3个可视模式子集： - Visual [Character]模式: 用于选择单个字符 - 视觉线模式: 用于一次选择整条线 - 可视块模式: 用于选择块状矩形形状的文本

    1.2.2 切换到可视模式

    如果要切换到可视模式，则需要先进入普通模式: - 视觉模式(VISUAL) v(小写) - 视觉线模式(VISUAL LINE) V(大写) - 视觉块模式(VISUAL BLOCK) Ctrl + v

    1.2.3 使用可视模式

    当切换到可视模式时，Vim会给你一些细节，它将显示当前处于哪种模式以及选择的文本数量。进入可视模式后，选择从光标点开始，使用导航键[h/j/k/l]移动它。当你对选择感到满意时，可以执行删除、拉取和替换等操作。

    vim快捷键

    复制: v + y + p 剪切: v + d + p gg: 将光标移动到文档的开头 G: 将光标移动到文档的末尾
